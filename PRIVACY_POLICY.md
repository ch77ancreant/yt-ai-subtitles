# 隱私權政策（Privacy Policy）

**擴充功能名稱：** 日/韓語AI翻譯字幕
**最後更新日期：** 2026-07-05

感謝您使用 "日/韓語AI翻譯字幕"。本政策說明本擴充功能在執行過程中會接觸到哪些資料、如何處理這些資料，以及您可以如何控制它們。

## 1. 這個擴充功能做什麼

"日/韓語AI翻譯字幕" 會在您觀看 YouTube 影片時，攔截該影片的韓文或日文字幕（CC）資料，將內容切成小段，傳送給您自行設定的翻譯引擎（Gemini / ChatGPT / DeepSeek 三者之一）進行翻譯，再把原文（日文／韓文）與繁體中文的雙語字幕疊加顯示在影片畫面上。

## 2. 我們會接觸到哪些資料

- **YouTube 字幕內容**：僅限您正在觀看的影片的日文或韓文字幕文字內容（時間軸與文字），用於翻譯與顯示，不會包含您的個人身份資訊。
- **API Key**：您在擴充功能的設定（Popup）中輸入的 Gemini / ChatGPT / DeepSeek API Key。
- **開關狀態與所選翻譯引擎**：例如「是否啟用雙語字幕」「目前選用哪個翻譯引擎」等設定值。

本擴充功能**不會**蒐集您的瀏覽紀錄、帳號資訊、裝置識別碼，也**沒有**任何分析（analytics）或追蹤（tracking）程式碼。

## 3. 資料如何儲存

所有設定（包含 API Key）都只會儲存在您瀏覽器本機的 `chrome.storage.local` 中，**不會**被上傳到開發者或任何第三方伺服器。移除此擴充功能即會清除這些本機儲存的資料。

## 4. 資料如何傳輸給第三方

為了完成翻譯，字幕文字會直接從您的瀏覽器，透過您自己的 API Key，傳送到您所選擇的翻譯服務供應商：

- Google（`generativelanguage.googleapis.com`）
- OpenAI（`api.openai.com`）
- DeepSeek（`api.deepseek.com`）

這個傳輸是直接從使用者瀏覽器發出（peer-to-peer 對該服務商），開發者本人或本擴充功能的伺服器**不會經手、不會留存**任何字幕內容或翻譯結果。您傳送給上述服務商的內容，會依各服務商自身的隱私權政策與服務條款處理，建議您自行查閱：

- Google：https://policies.google.com/privacy
- OpenAI：https://openai.com/policies/privacy-policy
- DeepSeek：https://www.deepseek.com/privacy

## 5. 為什麼需要這些權限

- `storage`：用於儲存您的開關設定、所選翻譯引擎與 API Key。
- `host_permissions`（`generativelanguage.googleapis.com` / `api.openai.com` / `api.deepseek.com`）：用於讓擴充功能能直接向您選用的翻譯引擎發送翻譯請求。
- 在 YouTube 網域上執行的內容腳本：用於讀取字幕資料並在畫面上疊加顯示雙語字幕。

本擴充功能不會請求超出上述用途所需的任何權限。

## 6. 您的選擇與控制權

- 您可以隨時在擴充功能的設定中關閉雙語字幕功能。
- 您可以隨時清除或更換已輸入的 API Key。
- 移除擴充功能即會清除所有本機儲存的設定資料。

## 7. 政策更新

若本政策有重大變更，將會更新本檔案中的「最後更新日期」。建議您在更新擴充功能版本後，重新確認本政策內容。

## 8. 聯絡方式

若您對本隱私權政策有任何問題，歡迎透過 Chrome 線上應用程式商店的開發者聯絡資訊與我們聯繫。
